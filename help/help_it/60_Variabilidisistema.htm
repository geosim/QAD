Variabili di sistema

Le variabili di Sistema sono delle impostazioni che controllano il comportamento di alcuni comandi. Possono essere di tipo intero, reale, carattere, booleano or colori RGB (es. “#FF0000”). Se esiste un progetto corrente, sono salvate e caricate nel file <nome progetto corrente>_QAD.INI della cartella del progetto corrente altrimenti verrà usato il file QAD.INI situato nella cartella di installazione.
